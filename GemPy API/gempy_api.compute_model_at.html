gempy_api.compute_model_at¶

gempy_api.compute_model_at(new_grid: numpy.ndarray, model: gempy.core.model.Model, **kwargs)[source]¶

This function creates a new custom grid and deactivate all the other grids and compute the model there:

This function does the same as compute_model() plus the addition functionallity of

passing a given array of points where evaluate the model instead of using the gempy.core.data.GridClass.
