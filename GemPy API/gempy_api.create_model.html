Create a Model object

Model

(inserted)

Container class of all objects that constitute a GemPy model.

In addition the class provides the methods that act in more than one of this class. Model is a child class of

DataMutation and MetaData.

(inserted) Class containing metadata of the project.

Set of attributes and methods that are not related directly with the geological model but more with the project

Args:

project_name (str): Name of the project. This is use as default value for some I/O actions

Attributes:

date (str): Time of the creations of the project

project_name (str): Name of the project. This is use as default value for some I/O actions

This class handles all the mutation of an object belonging to model and the update of every single object depend

on that.

Attributes:

grid (Grid): [s0] -(inserted)- Class to generate grids.

faults (Faults): [s1] -(inserted)- Class that encapsulate faulting related content. Mainly, which surfaces/surfaces are faults. The fault network—i.e. which faults offset other faults—and fault types—finite vs infinite.

series (Series): [s2] -(inserted)- Class that contains the functionality and attributes related to the series. Notice that series does not onlyrefers to stratigraphic series but to any set of surfaces which will be interpolated together (comfortably).

surfaces (Surfaces): [s3] -(inserted)- Class that contains the surfaces of the model and the values of each of them.

surface_points (SurfacePoints): [s4] -(inserted)- Data child with specific methods to manipulate interface data. It is initialize without arguments to giveflexibility to the origin of the data.

orientations (Orientations): [s5] -(inserted)- Data child with specific methods to manipulate orientation data. It is initialize without arguments to giveflexibility to the origin of the data.

rescaling (Rescaling): [s6] -(inserted)- Auxiliary class to rescale the coordinates between 0 and 1 to increase float stability.

additional_data (AdditionalData): [s7] -(inserted)- Container class that encapsulate Structure, KrigingParameters, Options and rescaling parameters

interpolator (InterpolatorModel): [s8] -(inserted)- Child class of Interpolator which set the shared variables and compiles the theanograph to compute the geological model, i.e. lithologies.
